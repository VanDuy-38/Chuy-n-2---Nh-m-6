## 🎯 Mục Tiêu Dự Án
- Xây dựng ứng dụng đặt sân di động mượt mà, thân thiện với "lông thủ".
- Hiển thị tình trạng sân trống theo thời gian thực (Real-time availability) để tránh trùng lịch.
- Số hóa quy trình quản lý sân bãi, giúp chủ sân nắm bắt lịch đặt và doanh thu dễ dàng.
- Tự động hóa quy trình đặt cọc, thanh toán và nhắc lịch.

## ✨ Tính Năng Chính
- 🔐 **Quản lý tài khoản:** Đăng ký, đăng nhập và phân quyền (Người chơi, Chủ sân, Admin).
- 🏟️ **Tìm kiếm & Khám phá:** Xem danh sách cụm sân cầu lông, lọc theo khu vực, giá tiền và tiện ích.
- ⏰ **Tra cứu lịch trống:** Xem chi tiết lưới thời gian (time-slots) của từng sân trong ngày.
- 🛒 **Đặt sân & Quản lý lịch đá:** Chọn khung giờ, đặt sân và theo dõi trạng thái đơn (Chờ duyệt, Đã cọc, Thành công).
- 💳 **Thanh toán:** Tích hợp thanh toán tiền cọc/toàn bộ (giả lập hoặc qua ví điện tử).
- ⭐ **Đánh giá:** Chấm điểm chất lượng mặt sân và dịch vụ sau khi trải nghiệm.
- 📊 **Dashboard Chủ sân:** Quản lý sân, khung giờ, linh hoạt điều chỉnh giá (giờ vàng/thường) và thống kê doanh thu.

## 🛠️ Công Nghệ Sử Dụng
- **Frontend (Mobile & Web App):** Flutter
- **Backend (API):** Laravel (PHP)
- **Database:** MySQL / PostgreSQL
- **Authentication:** Laravel Sanctum / JWT
- **API:** RESTful API
- **Deployment:** VPS / Docker
- **Version Control:** Git
